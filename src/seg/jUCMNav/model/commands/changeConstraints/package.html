Commands to move/resize model elements.

Package Specification

The commands contained in this package move/resize model elements.

  • It is to be noted that SetConstraintComponentRefCommand should not be used outside the package; use SetConstraintBoundComponentRefCompoundCommand instead.
  • It is to be noted that MovePathNodeCommand should not be used outside the package; use SetConstraintCommand instead.
