EditParts used by the Scenario/Strategy view.

Package Specification

EditParts are GEF specific; we recommend you read some GEF documentation before playing with these classes. See DevDocEditPartCreation to understand how EditParts are created and DevDocOutline to see how the outline works. The scenario/strategy view is a TreeView; our elements are therefore TreeEditParts.

  • The edit part factory is StrategyTreeEditPartFactory.
  • Our root edit part is StrategyRootEditPart
  • All model elements use StrategyUrnModelElementTreeEditPart
