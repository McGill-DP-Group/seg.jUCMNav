XYLayout EditPolicies used by seg.jUCMNav.editparts

Package Specification

EditPolicies are GEF specific; we recommend you read some GEF documentation before playing with these classes. See DevDocEditPolicies to learn more about them.

XYLayout EditPolicies are used by to add elements to the map using the palette and drag&drop of PathNodes on NodeConnections or other PathNodes. Furthermore, the XYLayoutEditPolicy used by the MapAndPathGraphEditPart manages moving and resizing of PathNodes and other elements.

All of the classes in this package are very important and define jUCMNav's behaviour.
