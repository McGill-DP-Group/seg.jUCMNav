Resource management for jUCMNav.

Package Specification

UCMNavMultiPageEditor delegates to MultiPageFileManager for create(), doSave() and doSaveAs(). The latter does editor related work and delegates to UrnModelManager for the low level XMI serialization. ResourceTracker is a file system listener hooked to the UCMNavMultiPageEditor.

Also contains classes to modify the new scenarios metamodel (ScenarioSpec) in a similar fashion to the URNspec.
